* 7 月 9 日，摩纳哥成为全球第一个实现5G全国覆盖的国家，这源于该国运营商摩纳哥电信与华为的深入合作。
摩纳哥公国位于法国南部，是个典型的微型国家，国土面积仅仅 2 平方公里，但拥有近 4 万人口，是世界上人口最稠密的国家之一，同时也是世界上人均收入最高的国家之一。
